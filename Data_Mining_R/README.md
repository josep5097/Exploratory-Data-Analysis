# Exploratory-Data-Analysis
> Exploratory Data Analysis with R

Se han realizado los procesos de Data Mining a diferentes datasets.
-   En la carpeta Mod1 se trabajó la introducción de la Minería de datos.
-   En la carpeta Mod2 se trabajó la parte del preprocesamiento de los datos.
-   En la carpeta Mod3 se trabajó los modelos de Regresión Logística.
-   En la carpeta Mod4 se trabajó sobre los indicadores para evaluar los modelos de clasificación.
-   En la carpeta Mod5 se trabajó sobre las clases desbalanceadas.
-   En la carpeta Mod6 se trabajó sobre el algoritmo de k-nn y Naive Bayes
